Flex-box Utilities

.u-flex          => Sets display to flex
.u-flex-x        => Sets flex-direction to row
.u-flex-center-x => Centers flex item on X-Axis
flex item
.u-flex          => Sets display to flex
.u-flex-y        => Sets flex-direction to column
.u-flex-center-x => Centers flex item on X-Axis
flex item

Note: .u-flex-y Sets the flex direction to column what that means is that it aligns flex items vertically. Flexbox does that by inverting the Axes. If we want to center flex item vertically then we'd use .u-flex-center-x.

Important: You can still use .u-flex-center-x .u-flex-center-y without setting the flex-direction in a specific direction!
